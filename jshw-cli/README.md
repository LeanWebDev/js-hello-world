# Usage
<!-- usage -->
```sh-session
$ npm install -g jshw-cli
$ jshw-cli COMMAND
running command...
$ jshw-cli (-v|--version|version)
jshw-cli/0.0.0 linux-x64 node-v14.17.1
$ jshw-cli --help [COMMAND]
USAGE
  $ jshw-cli COMMAND
...
```
